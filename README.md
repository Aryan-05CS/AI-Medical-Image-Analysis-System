# 🧠 AI-Powered Medical Image Analysis System

🚀 A deep learning-based web application that analyzes chest X-ray images to detect **Pneumonia** using transfer learning and explainable AI (Grad-CAM).

---

## 📌 Overview

This project simulates a real-world **AI healthcare system** that assists doctors in diagnosing diseases from medical images.

The system:
- Takes a chest X-ray as input  
- Uses a trained deep learning model  
- Predicts whether the patient is **NORMAL** or has **PNEUMONIA**  
- Visualizes model attention using **Grad-CAM**

---

## 🎯 Problem Statement

Manual analysis of medical images:
- is time-consuming  
- requires expert radiologists  
- is prone to human error  

This project aims to:
- assist in faster diagnosis  
- reduce workload on doctors  
- provide explainable AI insights  

---

## 🧠 Key Features

✅ Medical image classification (Pneumonia detection)  
✅ Transfer Learning using MobileNetV2  
✅ Grad-CAM visualization (Explainable AI)  
✅ Interactive web interface (Streamlit)  
✅ Real-world dataset simulation  
✅ Confidence score output  

---

## 🛠️ Tech Stack

- **Programming:** Python  
- **Deep Learning:** TensorFlow / Keras  
- **Computer Vision:** OpenCV  
- **Visualization:** Matplotlib  
- **Web App:** Streamlit  
- **Model:** MobileNetV2 (Transfer Learning)
